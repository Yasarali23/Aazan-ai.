## 🎯 3 तरीके से Download कर सकते हो

## METHOD 1: Direct Download (सबसे आसान) ⭐

### Step 1: Browser में जाओ
```
Ye page open है अभी
जहाँ तुम message भेज रहे हो
```

### Step 2: "Files" या "Outputs" Tab ढूंढो
```
Top में देखो
कोई "Files" या "Outputs" या "Download" button होगा
```

### Step 3: सभी Files Select करो
```
Ctrl+A दबाओ (सब files select हो जाएंगी)
या individually click करो
```

### Step 4: Download करो
```
Right click करो
"Download" चुनो

या

आपको एक ZIP file मिलेगी automatically
```

### Step 5: Extract करो
```
Windows:
- Right click → Extract All

Mac:
- Double click (automatically extracts)

Linux:
- unzip filename.zip
```

## METHOD 2: Individual File Download

### Step 1: हर File को देखो
```
सभी files की list दिख रही है
```

### Step 2: एक एक करके Download करो
```
हर file के आगे Download button है

```

```
एक नया folder बनाओ: aazan-ai
सभी files को इसमें रखो
```

## METHOD 3: Copy-Paste करो

### अगर Direct Download नहीं हो रहा तो:

#### Step 1: File का content copy करो
```
File को open करो
Ctrl+A करो (सब select)
Ctrl+C करो (copy)
```

#### Step 2: अपने computer पर file बनाओ
```
Notepad/VSCode खोलो
Content paste करो
Save करो
```

#### Step 3: सभी files के लिए दोहराओ
```
हर file के लिए यह करो
```

## 💻 Desktop पर Download करो

### Windows:
```
1. ये page खोलो
2. Ctrl+S दबाओ (Save Page)
3. या हर file पर Right click → Save
4. Folder choose करो
5. Files save हो जाएंगी
```

### Mac:
```
1. ये page खोलो
2. Command+S (Save)
3. या Right click → Save Link As
4. Folder choose करो
```

### Linux:
```
1. Firefox/Chrome खोलो
2. Right click → Save Link As
3. /home/username/Downloads में save करो
```
